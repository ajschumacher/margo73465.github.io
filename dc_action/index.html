Demographic Visualization for DC Action

DC Action for Children is a Washington DC based non-profit that aims to use "data-based analysis ... to promote policies and actions that optimize child and family well-being." Through the data scientist volunteer organization DataKind, I volunteered to assist in their "data-book" project. The graphic below is a much simplified version of the full data book (most updated version of the full project can be found here). I created the chart on the lower right hand side using D3 to replace a pair of pie charts that displayed the data in previous iterations of the project. Hover over differnt neighborhoods to see how the information changes.
